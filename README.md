# crosslink hub (v3 checkpoint)

This branch is the **machine-written** materialized state of the crosslink issue
tracker. It is regenerated by compaction after every mutation — do not edit it by
hand; manual changes are overwritten on the next compaction.

## What lives here

- `state.json` — the compacted [`CheckpointState`] (the authoritative cache the
CLI hydrates from).
- `issues/<uuid>.json` — one rendered file per live issue, comments and time
entries inline. Browse them directly in the GitHub web UI.
- `meta/milestones.json` — the milestone registry.

The append-only **event logs** that this state is reduced from live on the
per-agent branches `crosslink/agents/<agent-id>`. Each agent is the single writer
of its own branch.
