# Sources

External materials used during research. This directory catalogs the evidential foundation for EDASES research.

## Purpose

Store and reference the primary materials from which observations, findings, and assumptions are derived.

## Subdirectories

- `papers/` — Research papers and published studies
- `repositories/` — Repository reviews and codebase analyses
- `methodologies/` — Methodology analyses and process documents
- `communities/` — Community discussions, forum threads, and collaborative inputs
- `historical-projects/` — References to prior projects and legacy work

## Conventions

- Each source entry should include a brief description and link/reference where available.
- Sources should be tagged by type (paper, repository, methodology, etc.) and topic.
- Prefer linking to stable URLs when possible; archive local copies for critical references.
