# Evaluation Corpus

Artifacts used to evaluate candidate methodologies, architectures, and processes. This directory is research infrastructure.

## Purpose

Store concrete examples, cases, and test materials that allow EDASES methods to be validated against real-world scenarios.

## Examples

- Historical project documents
- Handoff examples (successful and failed)
- Context-loss examples
- Decision reconstruction examples
- Known failure cases
- Benchmark scenarios

## Conventions

- Each entry should include context about what it demonstrates
- Tag by evaluation purpose (benchmark, failure-case, success-case, regression)
- Preserve original formatting and structure of source materials
- Date entries to preserve temporal context
