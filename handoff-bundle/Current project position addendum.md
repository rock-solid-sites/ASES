# Current Project Position Addendum

## Research Before Tooling
Requirements and patterns should be identified before tooling decisions are made.

## Knowledge Architecture As Research Infrastructure
Knowledge architecture exists to address observed failures:
- Knowledge loss
- Context loss
- Assumption drift
- Decision provenance loss
- Handoff difficulties

## Methodology Scope
The architecture is intended to support a reusable methodology, not merely a single project.

## Current Architectural Assumption
Future users will primarily navigate relationships rather than documents.

## Current Priority
Knowledge architecture research and construction of an evaluation corpus.
