# Findings

Conclusions derived from one or more observations. Findings represent interpreted results rather than raw evidence.

## Purpose

Store conclusions that emerge when observations are analyzed, compared, or synthesized.

## Characteristics

- Derived from one or more observations
- Represent interpreted results, not raw evidence
- May be preliminary or validated
- Should reference the observations that support them

## Conventions

- Each finding should reference its supporting observation(s)
- Use descriptive filenames that convey the finding
- Date entries to preserve temporal context
- Tag by confidence level (preliminary, supported, validated)
