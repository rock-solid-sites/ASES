# Syntheses

Higher-level integration across multiple findings and sources. Syntheses represent the most mature form of analysis in the knowledge chain.

## Purpose

Store integrated analyses that combine multiple findings into coherent narratives, comparisons, or reviews.

## Examples

- State-of-the-art reviews
- Comparative analyses
- Literature syntheses
- Methodology comparisons
- Cross-domain pattern analyses

## Conventions

- Each synthesis should reference its constituent findings and sources
- Use descriptive filenames that convey the synthesis scope
- Date entries to preserve temporal context
- Tag by type (review, comparison, integration, analysis)
